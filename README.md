## ⚙️ Installation

```bash
# Clone repository
git clone https://github.com/your-username/WalletTown.git

# Masuk ke folder
cd WalletTown

# Install dependencies
npm install

# Jalankan project
npm run dev
```

---

## 🔑 Environment Setup

Buat file `.env` dan isi dengan konfigurasi berikut:

```
PORT=3000
DATABASE_URL=your_database_url
PRIVATE_KEY=your_private_key
API_KEY=your_api_key
```
